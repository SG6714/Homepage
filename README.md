## Homepage

Welcome to the source code for my personal website, an integration of advanced technology and modern personal branding. This project highlights my expertise as a technologist and underlines the seamless incorporation of technology infrastructure into everyday solutions.

The goal is to present a blueprint for creating a personal profile website while demonstrating how cutting-edge solutions like Docker, containerization, and modular designs can create efficient, scalable, and robust web platforms.

---

### Project Structure

```plaintext
├── app.py                # Backend logic for serving the website
├── docker-compose.yml    # Configuration for deploying services
├── dockerfile            # Instructions for building the Docker container
├── requirements.txt      # Dependencies and libraries for the Python environment
├── static/               # Directory for static assets, including CSS, JavaScript, and images
├── templates/            # Directory for HTML templates
├── README.md             # Documentation for the project
```

### Key Highlights

1. **Integration of Docker**: The application is containerized to ensure portability and consistency across different environments.
2. **Modular Architecture**: Separation of static files, templates, and backend logic for cleaner development practices.
3. **Blueprint for a Personal Profile Website**:
   - Use of technologies such as Python (Flask or Django backend) combined with modern containerization principles via Docker/Docker Compose.
   - Leverage VR-friendly frameworks like Three.js for innovative and immersive user experiences.

### Getting Started

To experiment or deploy the website, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/shaneguignard/Homepage.git
   cd Homepage
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Build and run Docker containers:
   ```bash
   docker-compose up --build
   ```
4. Access the site at `http://localhost:5000`.
